# 上海迪士尼乐园是否提供残障人士优惠票?

残障人士在购票时,可凭有效的残障证明享受优惠价格。如需咨询或购买,请致电上海迪士尼度假区预订服务中心(电话:400-180-0000 或 86-21-3158-0000)或前往上海迪士尼度假区官方手机应用程序(App)、上海迪士尼度假区官方网站、上海迪士尼度假区官方微信公众号购买。

残障人士入园时须携带本人有效身份证原件及本人有效残障证明原件,在主入口通过"中国残疾人联合会"微信公众号的"证件查询"⻚面验证本人的残疾证后方可入园。入园时无法出示本人的有效残障证明原件,或无法通过⻚面验证的游客,须按标准票的全价补全差价。

进入上海迪士尼乐园需要通过安检和检票口。排队安检时无包可以直接从安检⻔通行,带包的游客需要配合演职人员进行安全检查。

# 避免踩雷提示

## 食品携带规定

游客可携带供本人食用的食品及饮料进入上海迪士尼乐园。但禁止携带物品包括（但不限于）：
- 需加热、再加热、加工、冷藏或保温的食品及带有刺激性气味的食品
- 如需加热水食用的方便面
- 带自热功能的食品
- 西瓜（需要用刀切开的整个西瓜）
- 榴莲
- 臭豆腐等

## 禁止携带物品

- 易燃易爆物品（如鞭炮）
- 武器、仿真武器或玩具枪（包括玩具爆破器、水枪等）
- 酒精饮料
- 罐装或玻璃容器（小型婴儿罐装食品除外）
- 尺寸超过56厘米×36厘米×23厘米的箱包容器或行李
- 任何尺寸带轮子的行李箱
- 不论尺寸大小的以下载具或车辆（除儿童推车尺寸另有规定外），包括但不限于野营车、儿童野营推车、手推车、三轮或三轮以上儿童推车、带轮子的运送工具、含儿童座椅的行李箱、拖挂车类物品、长度超过110厘米、宽度超过85厘米的儿童推车
- 带滑轮的玩乐装备，如旱冰鞋及带滑轮的鞋子、滑板、踏板车
- 可能对他人造成安全隐患、妨碍乐园或任何相关设施运营或扰乱整体游园环境的照相机和摄像机以及包括但不限于以下种类的专业摄影摄像相关设备：所有三脚架、带有三脚架或任何延展伸缩功能的自拍杆和稳定器、任何形式的专业或商用稳定器、照明设备等
- 动物（导盲犬除外）。导盲犬须时刻栓有系绳并在主人的看管下。部分游乐项目也可能不允许导盲犬进入
- 相关法律禁止的物品及其他具有危险性或破坏性的物品

# 重返乐园政策

若计划暂时离开乐园并于当日重返乐园,请保存好乐园⻔票,在返回乐园时请在入园闸机处向演职人员出示本人有效⻔票即可再次入园。

## 友情提示:
- 请勿将⻔票与他人交换或混淆,错拿⻔票会导致无法再次入园
- 当日二次入园的入口除主入口外,也可以通过迪士尼小镇入口进入
- 请在乐园当日营运时间内重返乐园。重返乐园的开放时间以当天实际情况为准

# 常⻅问答

## 1. 可以退票吗？

通过官方直营渠道购买乐园标准⻔市价*"一日票/二日票",迪士尼妙游一日套票的游客,且在购票时未使用任何上海迪士尼度假区官方发放的优惠券,在购票后未进行过日期变更,可在以下规定的时间内通过原订单详情界面自助办理退票。

- 于到访日前7天的23:59(含)之前提交退票申请,可申请全额退票
- 自到访日前6天的00:00至到访日前1天的23:59(含)期间提交退票申请,每日每张⻔票将收取人⺠币80元退票服务费
- 到访日当日00:00之后不可退票

*"二日票"的退票标准以购票时所选定的首个到访日期为准。
*"乐园标准⻔市价",⻔票不包含任何用折扣或促销而购买的⻔票产品。

## 2. 身份证明原件丢失或入园当日忘记携带，怎么办？

预订相关问题请拨打400-180-0000或+86-21-3158-0000咨询。非预订相关问题,请拨打+86-21-2099-8001。或可以寻求乐园游客服务中心的帮助。身份证为重要的入园验证凭证,出行前请务必检查,以免延误入园时间或造成不便。

## 3. 买了票之后，随便哪天都能进园吗？

购买了指定日⻔票的游客,只能在票面上显示的特定日期内入园。

## 4. 乐园里的游乐项目还需要额外收费吗？

持有一张上海迪士尼乐园⻔票将可在乐园当天的运营时间内通行乐园内所有的游乐项目和娱乐演出,但园内的迪士尼乐拍通、迪士尼尊享卡、迪士尼礼宾服务以及迪士尼尊享导览等需额外付费。
